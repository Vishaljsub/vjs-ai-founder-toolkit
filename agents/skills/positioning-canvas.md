# Skill: Positioning Canvas

**Run by:** positioning-strategist · **Output to:** `../workspace/01-positioning/positioning-canvas.md`

Fill every box with specifics. No adjectives without a noun behind them.

| Box | Prompt | Fill in with validated answers |
|---|---|---|
| **1. Competitive alternatives** | What would the buyer use instead? | {{DIRECT_COMPETITORS}} · a DIY build · do nothing |
| **2. Unique attributes** | What only {{PRODUCT_NAME}} has | {{KEY_DIFFERENTIATORS}} |
| **3. Value** | The "so what" per attribute | translate each attribute into a buyer outcome sentence |
| **4. Best-fit customer** | Who values that MOST, urgently | {{BEACHHEAD_ICP}} |
| **5. Category frame** | The context that makes value obvious | pick the frame where you're the clear leader, not a follower |
| **6. Trend (optional)** | A wave you ride | a market/regulatory/technology shift that makes now the moment |

### Output: positioning statement template
> For **[beachhead ICP]** who **[job / trigger]**, **{{PRODUCT_NAME}}** is a **[category frame]** that **[primary value]**. Unlike **[main alternative]**, {{PRODUCT_NAME}} **[key differentiator]** — because **[reason to believe / proof]**.

### Done when
- One sentence, one beachhead, three defensible differentiators, and an explicit "not for" list.
